Vektordaten
Linien-/Kurven-Pfad: jeder Klick setzt einen weiteren Punkt (rastet am Grid ein). Abschließen per Doppelklick, Enter oder Button. Klick auf den Startpunkt schließt die Form direkt. Escape bricht ab.
Kreis/Ellipse: klicken und ziehen spannt das Begrenzungsrechteck auf. Shift gedrückt halten = perfekter Kreis. Entsteht als geschlossener Kurvenpfad, danach normal bearbeitbar.
Rechteck: klicken und ziehen spannt die Fläche auf. Shift gedrückt halten = perfektes Quadrat. Entsteht als geschlossener Linienpfad, danach normal bearbeitbar (auch einzelne Kanten in Kurven umwandelbar).
Stern/Zahnrad: Form (Stern/Zahnrad), Zacken/Zähne und Tiefe links einstellen, dann klicken und ziehen spannt das Begrenzungsrechteck auf. Shift gedrückt halten = gleichmäßige Proportionen. Entsteht als geschlossener Linienpfad, danach normal bearbeitbar.
Bearbeiten: Ankerpunkte (weiß) und Kontrollpunkte (orange) ziehen — rasten am Grid ein. Doppelklick auf ein Segment fügt dort einen neuen Punkt ein, Rechtsklick auf einen Ankerpunkt entfernt ihn wieder. Klick auf ein Segment (nicht auf einen Punkt) wählt es für die Umwandlung Linie ↔ Kurve aus. Einen offenen Endpunkt nah an einen anderen offenen Endpunkt ziehen (weißer Ring erscheint) verschmilzt beide — schließt so einen Pfad oder verbindet zwei Pfade zu einem.
Transformieren: Pfad anklicken, dann per Anfasser verschieben, skalieren oder drehen. Ecken = gleichmäßig skalieren, Rauten an den Kanten = nur X- bzw. nur Y-Richtung skalieren, Punkt oben = drehen. Umschalt+Klick fügt Pfade zur Auswahl hinzu/entfernt sie, Ziehen auf leerer Fläche zieht ein Auswahlrechteck auf — mehrere Pfade lassen sich so gemeinsam transformieren. Beim Loslassen wird wieder aufs Grid gerastet.
Perspektivisch verzerren: Pfad(e) anklicken, dann eine der vier Ecken des Rahmens einzeln ziehen — die Auswahl wird wie eine Fläche im Raum verzerrt (Fluchtpunkt-Effekt). Ziehen innerhalb des Rahmens verschiebt die ganze Auswahl.
